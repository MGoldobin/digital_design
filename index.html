Обзор <React />

React — это декларативная, эффективная и гибкая JavaScript-библиотека для создания пользовательских интерфейсов. Она позволяет вам собирать сложный UI из маленьких изолированных кусочков кода, называемых «компонентами».

Особенности и преимущества

  • JSX - это расширение синтаксиса JavaScript, которое позволяет использовать HTML-подобный синтаксис для описания структуры интерфейса.
  • Props и State - это объекты, которые содержат информацию, которая влияет на то, что увидим после рендера компонета. Однако Props передается при создании компонета, а State изменяется во время жизненного цикла компонета.
  • VDOM - это концепция программирования, в которой идеальное или «виртуальное» представление пользовательского интерфейса хранится в памяти и синхронизируется с «настоящим» DOM при помощи библиотеки, такой как ReactDOM. Этот процесс называется согласованием.

Популярные npm пакеты для работы с react

    Количество скчиваний: 172к

    Количество скчиваний: 7.3м

  • Название пакета: redux и react-redux

    Назначение: инструмент для управления глобальным состоянием

    Количество скчиваний: 7.1м и 5.1м

  • Название пакета: axios

    Назначение: HTTP-клиент на основе Promise для браузера и node.js

    Количество скчиваний: 25.8м

  • Название пакета: @material-ui/core

    Назначение: UI kit с компонентами React для более быстрой и простой веб-разработки.

    Количество скчиваний: 2.1м

    Назначение: тестирование JavaScript и React
